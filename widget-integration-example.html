‚öôÔ∏è Configuration Options

apiUrl (required): Your API base URL (e.g., 'http://localhost:8000')

welcomeMessage: Custom welcome message

autoOpen: Auto-open widget on page load (true/false)

üîß Integration Methods

Method 1: Script Tag (Recommended)

<!-- Add to your HTML <head> or before closing </body> --> <script src="path/to/jewellery-chatbot-widget.js"></script> <script> window.JewelleryChatbot.init({ apiUrl: 'http://localhost:8000', position: 'bottom-right', theme: 'blue' }); </script>

Method 2: Module Import

// If using module bundlers (Webpack, etc.) import JewelleryChatbot from './jewellery-chatbot-widget.js'; JewelleryChatbot.init({ apiUrl: 'http://localhost:8000', position: 'bottom-right', theme: 'blue' });

Method 3: Dynamic Loading

// Load widget dynamically const script = document.createElement('script'); script.src = 'path/to/jewellery-chatbot-widget.js'; script.onload = function() { window.JewelleryChatbot.init({ apiUrl: 'http://localhost:8000', position: 'bottom-right', theme: 'blue' }); }; document.head.appendChild(script);
